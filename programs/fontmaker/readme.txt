### makeCanvas.py
Used to make fonts for OneLife such as the ones you see in graphicsSource.

Usage: python makeCanvas.py [FORMAT] [FONTFILE] [FONTSIZE]
    format - 32_32 or 32_64
    fontfile - path to font relative to this
    fontsize - font size in pt or ( 3/4 * pixels )
    
### printTable.py
Print latin1 table to console

Usage: python printTable.py
Alternative: python printTable.py > files/custom_table.txt
