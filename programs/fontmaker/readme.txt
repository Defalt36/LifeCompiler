### makeCanvas.py
Used to make fonts for OneLife such as the ones you see in graphicsSource.

Usage: python makeCanvas.py [FORMAT] [OVERLAY] [FONTSIZE]
    format - 32_32 or 32_64
    overlay - a transparent file in files that will overlay the image after built
    fontsize - font size in pt or ( 3/4 * pixels )

### showCharacterTable.py
Print latin1 table to console

Usage: python showCharacterTable.py
Alternative: python showCharacterTable.py > files/custom_table.txt

### deleteSquaresFromCanvas.py
Used mostly to create overlays. Behavior can only be edited in a text editor.
